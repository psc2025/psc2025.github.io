3. Como se usa o sistema

O sistema será projetado para operar de forma autônoma, com interação mínima do usuário final.

Inicialmente, o administrador deverá realizar o cadastro de rostos autorizados por meio de uma interface simples, acessível via aplicativo ou navegador web.

Após essa configuração inicial, o sistema funcionará continuamente, realizando a leitura facial através de uma câmera instalada no ponto de entrada e tomando decisões automaticamente com base nos dados cadastrados.

Além disso, o usuário poderá acompanhar o histórico de acessos e receber notificações em tempo real em caso de tentativas de acesso não autorizadas.

4. Benefícios do sistema ao usuário

  • Segurança aumentada: bloqueia automaticamente tentativas de acesso de pessoas não reconhecidas.
  • Conveniência: dispensa o uso de chaves, senhas ou cartões, utilizando apenas o reconhecimento facial.
  • Autonomia e monitoramento remoto: permite monitoramento e alertas em tempo real mesmo à distância.
  • Escalabilidade: pode ser facilmente adaptado para ambientes maiores, mantendo a mesma base tecnológica.
